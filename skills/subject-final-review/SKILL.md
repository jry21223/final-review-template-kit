# 学科期末复习包生成

## 触发条件

当用户做了以下任何一件事时加载本 Skill：

- 上传样卷、回忆版真题、课件、复习提纲等材料，要求生成复习资料
- 说"帮我做期末复习包""出两套模拟卷""整理复习讲义"
- 提到具体课程名 + "复习""模拟""冲刺""期末"

**不要等用户把所有材料给齐才开始。** 用户可能分多次上传，先用已有材料做第一版，后续补充再迭代。

---

## 输入材料（任意组合）

| 类型 | 示例 |
|------|------|
| 样卷/真题 | PDF、图片、Word、扫描件、回忆版文字 |
| 课件/笔记 | PPT、Markdown、手写笔记截图 |
| 复习提纲 | 老师画的重点、知识点清单、章节大纲 |
| 课后题 | 教材习题、作业答案 |
| 额外约束 | 课程名、学校、考试方式（开/闭卷）、总分、时长、难度 |

**冲突处理：** 最新用户消息 > 最新上传文件 > 旧材料。

---

## 工作流程（7 步）

### Step 1：材料分析

从材料中提取课程元数据。提取不到的写 `未明确`，**不要编造**。

```
课程名称：{{填入}}
学校/学院：{{填入}} 或 未明确
专业/年级：{{填入}} 或 未明确
学年学期：{{填入}} 或 未明确
考试方式：闭卷 / 开卷 / 未明确
卷面总分：{{填入}} 分 或 未明确
考试时间：{{填入}} 分钟 或 未明确
```

### Step 2：样卷结构识别

**这是整份复习包的骨架，必须严格匹配。**

| 题型 | 题量 | 每题分值 | 小计 | 考察特点 |
|------|------|----------|------|----------|
| 例：单项选择 | 10 题 | 3 分 | 30 分 | 概念判断、快速计算 |
| 例：填空 | 5 题 | 3 分 | 15 分 | 公式应用、短计算 |
| 例：计算/简答 | 4 题 | 10 分 | 40 分 | 过程推导、步骤完整性 |
| 例：综合/论述 | 1 题 | 15 分 | 15 分 | 跨模块、建模、论证 |
| **合计** | | | **100 分** | |

> ⚠️ 两套模拟卷的题型、题量、分值结构必须和这张表一致。

### Step 3：考点矩阵

把样卷涉及的知识点整理成优先级矩阵：

| 模块 | 样卷怎么考 | 题型位置 | 难度 | 优先级 |
|------|-----------|---------|------|--------|
| 模块A | 计算/推导 | 大题 | 中等 | 🔴 必考 |
| 模块B | 概念判断 | 选择 | 基础 | 🟡 高频 |
| 模块C | 综合应用 | 最后大题 | 较难 | 🔴 必考 |
| 模块D | 了解背景 | 选择1-2题 | 基础 | 🟢 了解 |

优先级分四级：
- 🔴 **必考核心** — 大题必出，必须掌握公式+步骤+模板
- 🟡 **高频重点** — 选择/填空常见，熟悉变形即可
- 🟠 **可能考点** — 偶尔出现，会基础套用
- 🟢 **了解即可** — 知道概念边界，不深挖

### Step 4：生成复习知识点讲义

使用 `templates/final-review-notes-template.tex`，替换以下变量：

```latex
\renewcommand{\CourseName}{高等数学II}        % 课程名
\renewcommand{\SchoolInfo}{XX大学/数学学院}   % 或保持"未明确"
\renewcommand{\MajorInfo}{理工类/大一}
\renewcommand{\TermInfo}{2025-2026学年第二学期}
\renewcommand{\ExamType}{闭卷笔试}
\renewcommand{\ExamTime}{120分钟}
\renewcommand{\TotalScore}{100分}
\renewcommand{\AuthorName}{AI整理}
\renewcommand{\WatermarkText}{仅供个人复习使用｜请勿商用传播}
```

讲义必须包含以下 10 个板块（按顺序）：

1. **考试范围概览** — 课程主线（基础→核心→综合）
2. **题型与分值结构** — 和 Step 2 一致
3. **考点优先级** — 四级分类 + 考点矩阵
4. **模块化知识点** — 每个模块包含：
   - 考试地位（必考/高频/可能/了解）
   - 核心概念（考试语言，不抄教材）
   - 核心公式/规则 + 适用条件
   - 常见模型/结论速查表
   - 方法模板（5步：识别→公式→代入→计算→检查）
   - 典型例题（新题，不照搬样卷）
   - 易错点（错误表现 + 正确做法）
5. **高频题型方法模板** — 识别特征 + 固定步骤 + 常用公式 + 易错点
6. **公式速查表** — 模块 | 公式/规则 | 使用场景
7. **易错点清单** — 易错点 | 错误表现 | 正确做法
8. **七天复习计划** — Day1~Day7 每天干什么
9. **三天冲刺版** — 时间不够时的精简版
10. **考前速记清单** — 最后 30 分钟扫一遍的内容

### Step 5：生成两套模拟卷

使用 `templates/final-mock-exam-template.tex`，替换变量后生成：
- `{{课程名称}}_模拟卷一_含答案解析.tex`
- `{{课程名称}}_模拟卷二_含答案解析.tex`

**出题规则（红线）：**

| ✅ 必须做 | ❌ 禁止做 |
|-----------|----------|
| 严格匹配样卷题型、题量、分值 | 直接复制旧题 |
| 每道题都有答案 | 只改数字/变量名/选项顺序 |
| 大题有完整解析（思路→公式→步骤→答案→易错提醒） | 打乱选项顺序当新题 |
| 两套卷考点错开但难度相当 | 两套卷互相平行克隆 |
| 新编题目，保持样卷风格和难度 | 照搬教材例题改个数 |

**两套卷分配策略：**

| 知识点 | 模拟卷一怎么考 | 模拟卷二怎么考 |
|--------|---------------|---------------|
| 知识点A | 小题（选择/填空） | 大题（计算/论述） |
| 知识点B | 大题 | 小题 |
| 知识点C | 卷一重点考 | 卷二略过或小题 |
| 知识点D | 卷一略过 | 卷二重点考 |

### Step 6：LaTeX 编译

```bash
# 编译讲义（需要两遍生成目录）
xelatex final-review-notes-template.tex
xelatex final-review-notes-template.tex

# 编译模拟卷一
xelatex {{课程名称}}_模拟卷一_含答案解析.tex
xelatex {{课程名称}}_模拟卷一_含答案解析.tex

# 编译模拟卷二
xelatex {{课程名称}}_模拟卷二_含答案解析.tex
xelatex {{课程名称}}_模拟卷二_含答案解析.tex
```

> **字体问题：** 默认用 Noto CJK。如果编译报字体找不到，改成系统可用字体：
> ```latex
> \setCJKmainfont{Source Han Serif SC}   % 思源宋体
> \setCJKsansfont{Source Han Sans SC}    % 思源黑体
> ```
> 或 `SimSun`（宋体）、`SimHei`（黑体）、`FangSong`（仿宋）等。

### Step 7：质量检查（交付前必须过）

- [ ] 用了最新的样卷和用户指令
- [ ] 课程元数据已提取，无法确认的标了 `未明确`
- [ ] 模拟卷题型、题量、分值 = 样卷结构
- [ ] 两套卷总分正确
- [ ] 每道题都有答案
- [ ] 大题/计算题/论述题有完整解析
- [ ] 讲义是考点优先级组织，不是教材平铺
- [ ] 包含高频题型方法模板
- [ ] 包含易错点清单
- [ ] 包含复习计划（7天版 + 3天版）
- [ ] 没有直接复制旧题
- [ ] 没有只改数字/变量名/选项顺序
- [ ] 两套卷考点错开、难度相当
- [ ] PDF 能正常打开
- [ ] 文件名符合规范
- [ ] 无敏感信息（真实姓名、学号、手机号等）

## 答案解析标准

**选择题：**
- 正确选项
- 判断依据
- 排除理由（有需要时）

**填空题：**
- 最终答案
- 关键计算/规则/定义

**计算/程序/证明/论述题：**
1. 解题思路（识别题型 + 方法选择）
2. 关键公式/规则/算法/框架
3. 主要步骤
4. 最终答案
5. 易错提醒

## 常见坑

| 坑 | 怎么避免 |
|----|---------|
| 模拟卷和样卷结构不一致 | 先建结构表，写完卷子后逐题核对分值 |
| 两套卷太像（平行克隆） | 先建考点分配表，确保同知识点用不同题型考 |
| 讲义变成教材摘抄 | 每个知识点都问"样卷怎么考的"，按考法组织 |
| 照搬旧题只改表面 | 出完题后自查：能不能一眼看出改编自哪道旧题？能→重写 |
| 大题缺解析步骤 | 解析必须5要素齐全：思路→公式→步骤→答案→易错提醒 |
| 遗漏复习计划 | 讲义最后必须有7天版和3天版计划 |
| 水印含个人信息 | 模板默认用中性水印，私有流程再覆盖 |
| 编译报字体错 | 先检查系统字体，换成可用的 CJK 字体 |
| LaTeX 编译需要两遍 | 第一遍生成 .aux，第二遍生成目录和页码引用 |
| 多模块时 .tex 文件太长 | 可以用 `\input{}` 拆分子文件，但最终仍能编译为单个 PDF |

## 学科适配提示

| 学科 | 讲义侧重点 | 模拟卷侧重点 |
|------|-----------|-------------|
| 数学/物理 | 公式推导、适用条件、变量含义、特殊情形 | 计算过程完整性、单位、符号规范 |
| 计算机/编程 | 算法流程、数据结构、时间复杂度、边界条件 | 代码阅读、输出预测、调试、伪代码 |
| 英语/语言 | 语法规则、阅读技巧、写作模板、翻译陷阱 | 完形、阅读、翻译、写作题型 |
| 人文/社科 | 定义辨析、比较表格、论证模板、案例框架 | 名词解释、简答、论述、案例分析 |
| 经管/法 | 概念框架、计算模型、法条适用、关键词 | 案例分析、计算题、论述题 |
| 医学 | 机制流程、鉴别诊断、用药原则、影像特征 | 病例分析、选择题、简答题 |

1. 📖 考前复习知识点讲义
